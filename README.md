Для использования у вас должен быть установлен Python версии выше 3.10.
После клонирования репозитория используйте "pip install -r requirements.txt" в консоли вашего интерпретатора(лучше делать это в виртуальном окружении(venv)).
Успешно установив библиотеки по предыдущей инструкции откройте файл "config.txt" и впишите в него следующие значения:
Username бота (к примеру @example_bot) который будет отвечать за ответ пользователю,
Промпт который будет создавать "сцену" для бота, в которой бот будет играть роль обычного собеседника.
Оба параметра разделите на разные строки, в файле уже есть пример.
